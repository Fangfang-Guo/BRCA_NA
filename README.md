# BRCA_NA
This repository contains code and pipelines for comparative analysis of **Native American (NA)** and **Caucasian (CA)** breast cancer cohorts across multiple omics layers: DNA mutations, copy number variations (CNV), RNA-seq expression.



## üìÇ Repository Structure and Run Order

### 1. DNA Mutation Analysis
Scripts for annotating, converting, filtering, and comparing somatic mutations.

1. `1. vep_annotation.sh` ‚Äì Annotate VCF files using VEP  
2. `2. vcf2maf_batch_convert.sh` ‚Äì Convert annotated VCF files to MAF format  
3. `3. filter_and_merge_maf.R` ‚Äì Filter mutations and merge across cohorts  
4. `4. fisher_test_and_volcano_plot.R` ‚Äì Compare mutation frequencies (Fisher‚Äôs test) and generate volcano plots  
5. `UMAP.R` ‚Äì Visualize mutation profiles using UMAP  



### 2. CNV Analysis
Scripts for CNV calling, filtering, statistical testing, and visualization.

1. `1. cnv_to_bed_and_annotation.R` ‚Äì Convert CNV calls to BED and annotate genes  
2. `2. filter_caucasian_cnv.R` ‚Äì Filter Caucasian CNV calls  
3. `3. cnv_statistical_analysis.R` ‚Äì Compare CNV frequencies across groups  
4. `4. cnv_volcano_plot.R` ‚Äì Generate volcano plots for CNV changes  
5. `5. cnv_results_tables.R` ‚Äì Summarize CNV results into tables  



### 3. RNA Sequencing Analysis
Scripts for downloading, processing, and analyzing RNA-seq data.

1. `1. download_rnaseq_caucasian.R` ‚Äì Download TCGA Caucasian RNA-seq data  
2. `2. process_na_rnaseq.R` ‚Äì Process Native American RNA-seq dataset  
3. `3. combine_rnaseq.R` ‚Äì Merge NA and CA RNA-seq matrices  
4. `4. differential_expression.R` ‚Äì Perform differential expression (DE) analysis  
5. `5. enrichment_analysis.R` ‚Äì Pathway enrichment based on DE genes  



### 4. Mutational Signature Analysis
Scripts for extracting and testing mutational signatures.

1. `1. mutational_signature_pipeline.py` ‚Äì Run SigProfilerExtractor for de novo mutational signature analysis  
2. `2. plot&test.py` ‚Äì Plot exposures and perform statistical tests  



### 5. Multi-Omics Pathway Analysis
Scripts for integrative analysis across DNA, CNV, and RNA.

- `multiomic_kegg_heatmap.R` ‚Äì Generate KEGG pathway heatmap integrating DNA, CNV, and RNA results
